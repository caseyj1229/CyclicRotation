# CyclicRotation
Given an array of integers A and an integer K, right-shift the array A a total of K times.  For example: [1,2,3,4,5] with a K value of 3 is [3,4,5,1,2].  [1,2,3,4,5] with a K value of 5 is [1,2,3,4,5].
